# build-deck v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade `claude-skills/build-deck` from scroll-only to dual-mode (Review / Present) with 4 new themes, theme dropdown, progress hairline, floating nav, on-slide page numbers, and a Terracotta/Carbon CSS bug fix.

**Architecture:** Single-file HTML template (`build-deck/template/index.html`) holds all CSS, JS, and rendering. All changes live in this file plus `SKILL.md`. State is tracked via `data-mode` and `data-theme` attributes on `<html>` and persisted to `localStorage`. Slides are kept in DOM in both modes; Present mode uses CSS to hide all but `.slide.is-current`.

**Tech Stack:** Vanilla HTML/CSS/JS. No build step, no test runner. js-yaml + DOMPurify (already loaded via CDN). Verification is manual via browser (gstack `browse` skill or any browser pointed at `file://`).

**Reference:** [Design doc](./2026-05-18-v2-design.md) · [Issue #1](https://github.com/LeoHChen/claude-skills/issues/1)

function injectPageNumbers() {
  const slides = document.querySelectorAll("#deck .slide");
  const total = slides.length;
  if (total === 0) return;
  const pad = String(total).length;
  slides.forEach((slide, i) => {
    let el = slide.querySelector(".slide__pageno");
    if (!el) {
      el = document.createElement("div");
      el.className = "slide__pageno";
      slide.appendChild(el);
    }
    el.textContent = `${String(i + 1).padStart(pad, "0")} / ${String(total).padStart(pad, "0")}`;
  });
}

document.addEventListener("keydown", e => {
  if (document.documentElement.dataset.mode !== "present") return;
  if (e.target.matches("input, textarea, [contenteditable], [contenteditable='true']")) return;
  switch (e.key) {
    case "ArrowLeft":
      showSlide(currentIndex - 1); e.preventDefault(); break;
    case "ArrowRight":
    case " ":
      showSlide(currentIndex + 1); e.preventDefault(); break;
    case "Home":
      showSlide(0); e.preventDefault(); break;
    case "End":
      showSlide(getTotal() - 1); e.preventDefault(); break;

  }
});
